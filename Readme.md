# Trainee Management API

## Technology Used
.net, asp.net core, c#, ef core

## How to Run
Clone the repo.

cd into the repo.

run the following command
    dotnet run

open browser http://localhost:5158

go to the "localhost:5158/swagger" to test all the implemented APIs

## API List

GET /api/health

GET /api/trainees
GET /api/trainees?searchParam=""
GET /api/trainees/{id}

POST /api/trainees/

PUT /api/trainees/

DELETE /api/trainees/
